# ECE 391: Computer Systems Engineering

**Spring 2025** | **University of Illinois Urbana-Champaign**

This branch contains coursework for ECE 391 - Computer Systems Engineering.

## Course Information

**Instructor:** Kirill Levchenko
**Architecture:** RISC-V (64-bit)
**Credits:** 4 credit hours

### Machine Problems

- **[MP1: RISC-V Assembly & Graphics](./ECE%20391/sp25_ece391_ky23-mp1/)** - Skyline screensaver in pure RISC-V assembly
- **[MP2: Device Drivers & Interrupts](./ECE%20391/sp25_ece391_ky23-mp2/)** - UART driver, VirtIO, kernel threading
- **[MP3: Operating System Kernel](./ECE%20391/sp25_ece391_group_24-mp3/)** - Complete OS kernel with processes, file system, virtual memory

## Course Topics

- Computer Organization & RISC-V Assembly
- Data Structures (queues, heaps, stacks, linked lists)
- Synchronization (mutexes, semaphores, race conditions)
- Interrupts & Exception Handling
- Virtual Memory & Paging
- Device Programming & Drivers
- File Systems (disk layout, inodes)
- I/O Interface & Memory Mapping
- Networking (sockets, protocols)
